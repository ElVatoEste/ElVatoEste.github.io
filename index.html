Agendador de Citas para la Biblioteca Pablo Antonio Cuadra

Proyecto desarrollado para la Biblioteca Pablo Antonio Cuadra de la Universidad Americana (UAM), con el objetivo de automatizar y gestionar las reservas de la sala privada.

Tecnologías: Java, Java Server Faces (JSF), XHTML, PrimeFaces, SQL Server, Apache Tomcat

Funcionalidades: Automatización de reservas, gestión de horarios, prevención de choques de fechas, integración con SQL Server para almacenamiento de datos. Incluye un exportador de reportes en Excel para facilitar la generación y análisis de datos.

Sistema Personal de Gestión Estudiantil

Sistema diseñado para permitir a los estudiantes categorizar apuntes, establecer metas, marcar eventos y organizar un cronograma cada semestre.

Tecnologías: Java, Java Server Faces (JSF), XHTML, PrimeFaces, SQL Server, Apache Tomcat, JavaScript

Funcionalidades: Gestión de apuntes y metas, marcado de eventos, organización de cronograma semestral. Utiliza JavaScript para mejorar la interactividad y experiencia del usuario.
